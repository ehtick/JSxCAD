# Orb(...dimentions)

Produces an orb that fits into the bounding box.

```JavaScript
Orb(5).and(Box(5, 5, 5).material('glass')).view().md('Orb(5)');
```

```JavaScript
Orb(3, 4, 5).and(Box(3, 4, 5).material('glass')).view().md('Orb(3, 4, 5)');
```
